# Screenshot archive

Historical screenshots from earlier NetSpeedTray versions (≈ v1.0.x → v1.2.5), kept here so the
progression of the UI over time isn't lost. **Nothing here is referenced by the current README** —
the live screenshots live in the parent `screenshots/` folder.

Roughly in version order:
- `main*.png` — the taskbar widget through the early releases
- `main_graph_*.png`, `graph.png`, `mini_graph.png` — the old standalone history-graph window (retired into the unified Monitor in 2.0)
- `settings_1.0.x` → `settings_1.2.5.png` — the Settings dialog as it evolved (fully redesigned for 2.0)

The last pre-2.0 shots still referenced by the v1.3.x README (`main_new_1.1.2.png`,
`main_graph_1.2.5.png`, `settings_1.3.3.png`) stay in the parent folder until the 2.0 README ships,
then they move here too.
